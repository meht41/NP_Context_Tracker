Русский:

Это расширение для Google Chrome позволяет быстро узнать статус отправления Новой Почты прямо на любой веб-странице. Просто выделите номер накладной (ТТН), кликните правой кнопкой мыши и выберите "Статус ТТН" в контекстном меню. Всплывающее уведомление мгновенно покажет актуальное состояние вашей посылки.
Расширение использует официальный API Новой Почты, поддерживает ввод и сохранение собственного API-ключа через настройки.

Возможности:

Проверка статуса отправления Новой Почты по номеру ТТН из любого текста на странице.
Быстрый доступ через контекстное меню.
Удобное уведомление о статусе посылки.
Настройка и сохранение собственного API-ключа.



English:

This Google Chrome extension allows you to instantly check the status of a Nova Poshta shipment on any web page. Simply select the tracking number (TTN), right-click, and choose "TTN Status" from the context menu. A popup notification will immediately display the latest status of your parcel.
The extension uses the official Nova Poshta API and lets you input and save your personal API key via the settings.

Features:

Check Nova Poshta shipment status by tracking number from any text on the page.
Quick access via context menu.
Convenient notification with parcel status.
API key input and storage for personalized access.
